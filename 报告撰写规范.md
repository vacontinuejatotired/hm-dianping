# 报告撰写规范

> 依据《报告撰写规范样张-排版要求》整理

---

## 一、题目与摘要

| 要素 | 格式要求 |
|------|----------|
| **论文题目**（封面/首页） | 三号黑体，居中；上下各空一行；尽量保持一行显示。 |
| **"摘要"二字** | 四号黑体，居中，上下各空一行。 |
| **摘要正文** | 小四号宋体，首行缩进 2 个汉字，约 300 字，1.5 倍行距。 |
| **"关键词"三字** | 四号黑体。 |
| **关键词内容** | 小四号宋体，以分号分隔。 |

---

## 二、正文总体格式

| 要素 | 格式要求 |
|------|----------|
| **正文字体** | 中文小四号宋体，英文小四号 Times New Roman；首行缩进 2 个汉字；1.5 倍行距。 |
| **一级标题**（如"3 I 级叶/盘转子错频方案的对比分析"） | 小三号黑体，居中，上下各空一行。 |
| **二级标题**（如"3.5 多自由度系统的强迫响应分析"） | 四号黑体，序数顶格写，空一格接标题，1.5 倍行距。 |
| **三级标题**（如"3.5.1 动态响应的计算方法"） | 小四号黑体，缩进两格书写序数，空一格接标题。 |
| **分项序号** | 对总项所包含的分项采用（1）、（2）、（3）……的序号。 |
| **公式转行** | 较长公式转行时优先在等号处转行，其次在 +、-、×、÷ 等符号处转行，符号写在行首；上下式尽可能在等号处对齐。公式编号（如 3-1）右对齐。 |
| **页脚页码** | 小五号宋体，居中。 |

---

## 三、图表格式

| 要素 | 格式要求 |
|------|----------|
| **图题** | 图序按章编号（如图 2.3），图题置于**图下方**，居中。图中曲线对应关系需在图注中说明。 |
| **表题** | 表序按章编号（如表 2-1），表题置于**表格上方正中**；表序写在表题左方不加标点，空一格接表题，表题末尾不加标点。表题用**五号宋体加黑**。 |
| **表格内容** | 中文用五号宋体，英文用五号 Times New Roman。 |
| **续表** | 表格跨页时，续表需重复表头，并在右上方写"续表 xx"（如"续表 3-1"）。 |
| **表格内单位** | 全表使用同一单位时，单位符号移至表头右上角。 |
| **表注** | 注置于表下方，用五号宋体。 |

---

## 四、参考文献

| 要素 | 格式要求 |
|------|----------|
| **"参考文献"标题** | 小三号黑体，居中，上下各空一行。 |
| **参考文献条目** | 按顺序编号，使用方括号 \[ \]。 |

**各类文献格式示例：**

| 类型 | 格式 |
|------|------|
| **图书** | [序号] 作者．书名[M]．出版地：出版社，年份：页码． |
| **期刊** | [序号] 作者．题名[J]．刊名，年份，卷号(期号)：起止页码． |
| **会议论文** | [序号] 作者．题名[A]．论文集名[C]．出版地：出版社，年份：页码． |
| **标准** | [序号] 标准编号，标准名称[S]． |
| **电子文献** | [序号] 作者．题名[EB/OL]．网址，日期． |
| **报纸** | [序号] 作者．题名[N]．报纸名，日期(版次)． |

---

## 五、附录

| 要素 | 格式要求 |
|------|----------|
| **附录标题**（如"附录 A 一维 1/f 波动数据和生成"） | 三号黑体，居中，上下各空一行。 |
| **附录中的代码** | 五号字体，单倍行距。 |
| **附录中的图** | 图序如"图 B1"，图题置于图下方。 |

---

## 六、其他细节

- **参考文献标注**：在正文中用上标的中括号标注，如 `[2]`、`[3, 6-9]`。
- **行距**：除特别说明（如代码单倍行距）外，正文、摘要、标题均为 1.5 倍行距。
- **缩进**：正文段落首行缩进 2 个汉字；三级标题缩进 2 格；二级标题顶格。
